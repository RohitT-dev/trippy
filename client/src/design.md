# CrewTravel Design Specification: "Horizon Bound" ## 1. Design Philosophy: The Elevated Explorer The CrewTravel aesthetic, "Horizon Bound," balances the excitement of discovery with the precision of AI curation. It uses a "Digital Concierge" approach—clean, airy, and high-contrast, ensuring that complex travel data feels organized and accessible. ## 2. Visual Foundation ### Color Palette * **Primary (Action):** `#0066FF` (Vibrant Blue) - Used for primary CTAs, active states, and key navigational elements. * **Surface/Background:** `#FFFFFF` (White) / `#F8FAFC` (Slate 50) - Provides a clean, professional canvas. * **Typography/Neutral:** * **Heading/Body High:** `#0F172A` (Slate 900) * **Body Medium:** `#475569` (Slate 600) * **Subtle/Border:** `#E2E8F0` (Slate 200) * **Semantic Colors:** * **Success:** `#10B981` (Emerald 500) * **Warning:** `#F59E0B` (Amber 500) * **Info:** `#3B82F6` (Blue 500) ### Typography * **Primary Font:** `Plus Jakarta Sans` * **Headings:** Semi-bold to Bold (weights 600-800), tight tracking (-0.02em), Slate 900. * **Body:** Medium weight (500) for better legibility on high-density screens. * **Captions/Labels:** Small caps or 12px Medium for metadata. * **Secondary Font:** `Inter` (used specifically for footer and technical metadata). ### Iconography & Shapes * **Corner Radius:** `8px` (Round Eight) standard for buttons, cards, and input fields. * **Icon Style:** Solid/Filled icons for primary actions; Outline icons for secondary navigation. * **Shadows:** Soft, low-blur shadows (`shadow-sm`) to define elevation without adding visual noise. ## 3. Shared Components ### Top Navigation Bar * **Layout:** Fixed top, full width, 64px height. * **Styling:** 80% white background with `backdrop-blur-xl`. * **Elements:** Brand logo (Left), Nav Links (Center), Primary CTA "Plan a Trip" (Right). ### Side Navigation (Booking Flow / Itinerary) * **Width:** 256px (64 units). * **Active State:** White card background with subtle shadow, bold text. * **Inactive State:** Slate 500 text, subtle hover shift to Slate 900. ### Activity/Feature Cards * **Structure:** Image (Top/Left), Content (Right/Bottom). * **Typography:** Bold titles, 14px body text, 12px metadata labels. * **Interactions:** Hover state includes a subtle scale up (1.02x) or shadow deepening. ## 4. Design Patterns ### AI Discovery Search * **Style:** Large, centered input field with an "AI Sparkle" icon. * **Prompting:** Encourages natural language (e.g., "A 10-day adventure through Japan..."). ### Step-by-Step Forms (Trip Configuration) * **Structure:** Numbered sections (01, 02...) with clear labels. * **Input Style:** Large, light-gray backgrounds (`bg-slate-100`) with generous padding. ### Map Integration * **Usage:** Used in "Interactive Itinerary" and "Hotel Selection." * **Styling:** Modern, minimal map tiles with custom markers in the primary blue or category-specific colors. ## 5. Interaction States * **Buttons:** * *Default:* Primary blue. * *Hover:* Deepened blue with transition. * *Active:* Scale 0.95. * **Cards:** Hover triggers `shadow-md` and `translate-y-[-2px]`. * **Inputs:** Focus state adds a 2px blue ring/border.
